KINDLE COVER ANALYZER - CREDENTIALS SETUP
===========================================

Copy the text below and save it as a file named ".env" (with the dot at the beginning):

OXYLABS_USERNAME=your_oxylabs_username
OXYLABS_PASSWORD=your_oxylabs_password
APIFY_TOKEN=your_apify_token
PORT=3000

INSTRUCTIONS:
1. Copy the 4 lines above (starting with OXYLABS_USERNAME...)
2. Create a new file called ".env" in this same folder
3. Paste the 4 lines into that file
4. Replace "your_oxylabs_username" with your actual Oxylabs username
5. Replace "your_oxylabs_password" with your actual Oxylabs password
6. Save the file

Your actual .env file should look something like:
OXYLABS_USERNAME=john@email.com  
OXYLABS_PASSWORD=mypassword123
APIFY_TOKEN=your_apify_token
PORT=3000
